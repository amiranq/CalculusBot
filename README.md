# CalculusBot

Telegram-бот для помощи в подготовке к экзаменам по математическому анализу.  
Проект принимает текст задачи или фотографию примера, после чего помогает определить, какие темы, определения и теоремы нужно знать для решения.

## Описание проекта

Основная цель проекта — упростить подготовку к решению задач по математике и анализу.

Бот поддерживает два сценария работы:
- пользователь отправляет текст задачи;
- пользователь отправляет фотографию примера.

После этого бот:
- анализирует текст задачи;
- ищет релевантные теоремы и определения в локальной базе;
- при работе с изображением отправляет фото в внешнюю LLM/API для анализа;
- возвращает пользователю список тем и теоретических материалов, которые пригодятся для решения.

Дополнительно в проекте есть команда с мотивирующими изображениями для пользователя.

## Основные возможности

- обработка текстовых сообщений;
- обработка изображений с задачами;
- поиск релевантных теорем по тексту задачи;
- интеграция с Telegram Bot API;
- интеграция с GigaChat API для анализа задач по изображению;
- выдача подсказок без полного решения задачи.

## Используемые технологии

- **pyTelegramBotAPI** — создание Telegram-бота;
- **Requests** — работа с HTTP-запросами и внешними API;
- **Pillow (PIL)** — обработка изображений;
- **scikit-learn** — TF-IDF и вычисление похожести текстов;
- **NLTK** — стемминг русского текста;
- **GigaChat API** — анализ изображений и генерация ответа;
- **Cosine Similarity** — поиск наиболее релевантных теорем;
- **TF-IDF Vectorizer** — представление текстов в виде числовых признаков.

## Структура проекта

- `main.py` — основной файл Telegram-бота;
- `API.py` — работа с внешним API для анализа задач;
- `file_exctrctor.py` — поиск подходящих теорем по тексту задачи;
- `calculusfile.txt` — локальная база теорем и определений;
- `Prom_images/` — изображения для дополнительной команды бота;
- `requirements.txt` — список зависимостей проекта.

## Принцип работы

### 1. Работа с текстом
Пользователь отправляет задачу текстом, после чего бот:
- нормализует текст;
- применяет стемминг;
- сравнивает запрос с базой теорем с помощью TF-IDF и cosine similarity;
- возвращает наиболее релевантные теоремы.

### 2. Работа с изображением
Пользователь отправляет фотографию задачи, после чего бот:
- скачивает изображение;
- сохраняет его локально;
- отправляет изображение во внешний API;
- получает ответ с перечнем тем, определений и теорем;
- отправляет результат пользователю в Telegram.
